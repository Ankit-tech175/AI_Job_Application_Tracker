# AI-Powered Job Application Tracker

Production-quality AI-powered job tracking system built using Flask, NLP, and Machine Learning.

## Features

- JWT Authentication
- ATS Match Scoring
- Resume Parsing
- Job Tracking
- Analytics Dashboard
- AI Skill Matching

## Tech Stack

- Flask
- SQLAlchemy
- spaCy
- scikit-learn
- Bootstrap 5
- Chart.js
